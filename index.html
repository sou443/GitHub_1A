Mac GitHubセットアップ

ユーザー名・メールアドレスを設定

git config --global user.name "ユーザー名" git config --global user.email "メールアドレス"

""の部分を入れなければ設定したユーザー名やメールアドレスを見れる

GitHubにSSHの設定をする

新しい SSH キーを生成する

ls ~/.ssh

id_〇〇.pubというファイルが１つも表示されない場合

ssh-keygen -t ed25519 -C "GitHubに登録したメールアドレス"

GitHub アカウントへSSHキーを追加する

pbcopy < ~/.ssh/id_ed25519.pub

SSHでGitHubに接続

ssh -T git@github.com

セットアップ後の使い方

1 変更をステージング

git add .

2 変更を記録

git commit -m "変更内容の説明"

3 Git Hubにアップロード

git push
通常フォルダファイルの確認 ls 隠しフォルダファイルの確認 ls -a フォルダ移動 cd ***** (ファイルの場所) mkdir ***** (ファイル名)
git init git add ファイル名 git commit -m 'メッセージ' git push -u origin main addの取り消し(ファイルの追加) git rm --cached ファイル名 addの取り消し(ファイルの変更) git restore --staged ファイル名 ローカル作業の取り消し(最後にコミットした状態に復元) git stash
